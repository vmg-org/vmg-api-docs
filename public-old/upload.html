Direct uploading to cloud storage. Cloud storage is secured and required a specific sign for authorization. This client makes two requests during file upload process:

  1. Get a signed url from a web server (check an auth user, create a sign from an access key, secret key and other parameters)
  2. Upload a file to this url (directly to cloud storage). The url is valid only 10 seconds after received.

Please select a file (upload process begins immediately)
